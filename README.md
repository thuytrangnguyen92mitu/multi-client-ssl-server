# multi-client-ssl-server
* Xây dựng mô hình Multi-Client Server và áp dụng SSL/TLS để nâng cao bảo mật cho hệ thống
    * Một GUI Server đơn giản có các chức năng cơ bản:
        - Start/Stop (listen),
        - Send 1 broadcast message đến tất cả clients đang có kết nối với nó.
    * Một GUI Client đơn giản với các chức năng:
        - Connect và Disconnect với server.
        - Nhận và hiển thị thông báo (broadcast message) từ Server.
    * Cả Server và Client đều phải áp dụng bảo mật SSL.
